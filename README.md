Lost & Found Ionic App
=====================

Lost & Found project for [ionic](http://ionicframework.com/ "Title")

## Using this project

Make sure the `ionic` utility is installed:

```bash
$ sudo npm install -g ionic
```
Install `bower` and its dependencies

```bash
$ npm install -g bower
$ bower install --save
```

Install `gulp` and run it

```bash
$ npm install -g gulp
$ gulp
```
